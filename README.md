WanderLust 🌍🏡

A full-stack travel accommodation booking web application inspired by Airbnb, where users can explore stays, create listings, and manage bookings with a modern and responsive interface.

🚀 Live Demo

https://wanderlust-project-kacp.onrender.com/listening

📌 Features
👤 User Authentication
Secure user signup and login
Password hashing and session management
Role-based access (Host / User)
🏠 Listings Management
Add new property listings
Upload images for listings
Edit and delete own listings
Detailed listing page with property information
🔍 Search & Explore
Browse all available stays
Search listings by location/title
Filter listings based on category or price
📅 Booking System
Reserve accommodations
Manage bookings
View booking details
⭐ Reviews & Ratings
Users can add reviews and ratings
Display average ratings on listings
📱 Responsive Design
Fully responsive UI for desktop, tablet, and mobile devices
